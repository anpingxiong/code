二叉查找树。
       为什么需要出现二叉查找树？因为当需要查找的数据不是有顺序的话，那么将他们构建成二叉查找树，将会提高查找效率。
       
       二叉查找树是在数据输入时构建的， 而不应该特意将所有的数据的转换为二叉树在去查询。 这样会导致多余的数据。
       
       
       二叉查找树应该提供的功能：
         insert  插入节点
         delete  删除节点
         search  搜索 
         max     拿到树的最大值
         min     拿到树的最小值
         successor 拿到某一个节点的后继
         predecessor 拿到某一个节点的前驱
